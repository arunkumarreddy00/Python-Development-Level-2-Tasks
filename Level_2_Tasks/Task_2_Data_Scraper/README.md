# Task 2: Data Scraper

## Description
A Python web scraper that extracts quotes and author names from a website.

## Features
- Uses requests library
- Uses BeautifulSoup for HTML parsing
- Extracts quote and author data
- Saves data into CSV file
- Handles request errors

## Requirements
```bash
pip install requests beautifulsoup4
```

## How to Run
```bash
python data_scraper.py
```
